Ambiqua App

Aplicación web progresiva desarrollada Full-Stack para monitorear y gestionar cultivos en tiempo real. Integra sensores físicos (ESP32) para medir humedad y temperatura del suelo, y permite visualizar los datos mediante gráficas interactivas con Chart.js. Incorpora panel de login, dashboard administrativo, gestión de usuarios y cultivos, subida de archivos y control automático/manual del sistema de riego mediante MQTT. Tecnologías: Flask, Python, MySQL, HTML, CSS, JS, Tailwind.
